# {frontmatter.title}

PetitSuis és una eina d'automatització de processos. El nom del programa ve de "navalla suïssa" i de "petits programes" i pretén ser una eina d'ús intern i per tenir a mà el codi per implementar a futurs programes. Actualment el programa té vàries funcionalitats, algunes d'elles utilitzen APIs de IA, d'altres APIs de tota la vida i d'altres utilitzen llibreries i programes per tractar textos, imatges, vídeos i àudios. PetitSuis està programat amb Django.
